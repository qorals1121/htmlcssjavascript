css basic

style 적용시 특정 문구에 각각의 디자인으로 적용하고 싶을 때 id, class 사용

id = #
. = class

id의 이름은 중복으로 쓰면 안된다. class는 똑같은 거 여러개 사용해도 된다.
웬만하면 class로 사용하는 것이 좋고 id는 정말 특별하게 구분할 때만 사용하는 것이 좋다. (웬만해선 사용 x)

스타일 상속 시 <span> <i> <a> 태그는 상속이 먹지않는다.

구글 웹폰트에서 폰트를 고를 시 언어를 korean으로 설정하고, style 갯수가 많은것을 고르면 좋다. (두께가 여러개인 것)

장바구니처럼 폰트를 담고 link 대신 import로 가져와 카피한다. (link는 html파일, import는 css파일에 넣는다)
